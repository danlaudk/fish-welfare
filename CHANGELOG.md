# Changelog

All notable changes to this dataset will be documented in this file.

## [1.0.0] - 2026-02-02

### Initial Release

First public release of FWI ARA program data.

**Datasets included:**
- `water_quality.csv` - 11,433 measurements (June 2021 - January 2026)
- `enrolled_ponds_2026-02-02.csv` - 210 enrolled ponds snapshot
- `stocking_harvest.csv` - 1,461 stocking/harvest events
- `dropouts.csv` - 94 program dropouts

**Coverage:**
- 304 unique ponds across West Godavari (241) and Nellore (63) districts
- Date range: June 19, 2021 to January 27, 2026

**Notes:**
- 2 stocking/harvest records with invalid dates were excluded from the source data
- All pond IDs anonymized using deterministic hashing
